## Status

- Concepts and cheat sheet complete.

## Overview

This module focuses on practical engineering ethics and risk management for
computing systems. It provides decision frameworks, checklists, and applied
examples for fairness, privacy, transparency, safety, and governance.

## How to use this module

- Read concepts in order to build a shared vocabulary.
- Use the cheat sheet during design reviews and risk triage.
- Apply case studies as templates for incident analysis (added in next branch).
- Use the ethics toolkit for project artifacts (added in later branch).
