# Modules

The `modules/` directory is the curriculum layer of the repository.

Each module is a self-contained study area with a module README and a numbered concept sequence. The numbering inside `01-concepts/` reflects intended reading order rather than arbitrary file creation order.

## How To Use This Directory

- start with the module README for scope and navigation
- read numbered concept pages in order
- use cheatsheets, implementations, case studies, exercises, and notes as supporting layers
