# Concepts

Read these pages in numerical order. Later pages assume the modeling and evaluation discipline established by the earlier ones.

- [01 Supervised Learning Foundations](01-supervised-learning-foundations.md): the basic learning setup, target prediction, and loss-driven model fitting.
- [02 Feature Engineering and Preprocessing](02-feature-engineering-and-preprocessing.md): turning raw data into stable model inputs without leaking validation information.
- [03 Evaluation Metrics and Validation](03-evaluation-metrics-and-validation.md): the metrics and validation choices that determine whether model quality claims are trustworthy.
- [04 Data Leakage and Reproducibility](04-data-leakage-and-reproducibility.md): the controls that keep experiments honest and comparable across runs.
- [05 Regularization and Bias/Variance](05-regularization-and-bias-variance.md): the main generalization tradeoffs that explain underfitting and overfitting.
- [06 Unsupervised Learning Foundations](06-unsupervised-learning-foundations.md): clustering and structure discovery without labeled targets.
- [07 Dimensionality Reduction](07-dimensionality-reduction.md): lower-dimensional representations for denoising, compression, and visualization.
- [08 Loss Functions and Optimization for Machine Learning](08-loss-functions-and-optimization-for-machine-learning.md): the objectives and update rules that govern training behavior.
- [09 Model Selection and Hyperparameter Tuning](09-model-selection-and-hyperparameter-tuning.md): choosing among candidate models and search configurations without overfitting the validation process.
- [10 Class Imbalance and Calibration](10-class-imbalance-and-calibration.md): handling skewed label distributions and judging whether probabilities are trustworthy.
- [11 Deep Learning Training Loop](11-deep-learning-training-loop.md): batch training, gradients, updates, and validation monitoring for neural models.
- [12 Interpretability and Model Explanation Basics](12-interpretability-and-model-explanation-basics.md): global and local explanation tools for understanding model behavior.
- [13 Model Debugging and Error Analysis](13-model-debugging-and-error-analysis.md): a capstone workflow for diagnosing failure modes and prioritizing fixes.
