# Cryptography and Number Theory Exercises

These exercises should emphasize derivation, verification, and protocol reasoning rather than memorizing algorithm names.

## Suggested Practice

- compute gcd values, inverses, and modular exponentiation by hand
- explain which primitive matches a given security goal
- audit a toy protocol for missing authentication, validation, or key-management steps
