# Mini-Project: Navigation Controller Review

## Goal
Evaluate a simple navigation controller and document control tradeoffs.

## Steps
1. Run the pure pursuit test suite.
2. Use the Webots suite verification script (headless).
3. Summarize how parameter changes affect tracking error.

## Suggested Commands
- `python3 -m pytest -q modules/08-robotics-webots/03-implementations/python/tests`
- `bash modules/08-robotics-webots/03-implementations/webots-suite/scripts/verify_no_webots.sh`

## Success Criteria
- Tests pass with deterministic outputs.
- Report includes at least two parameter sensitivities.
