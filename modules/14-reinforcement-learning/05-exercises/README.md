# Reinforcement Learning Exercises

Use these exercises after the core concept sequence. The goal is not only to implement algorithms, but to practice debugging rewards, exploration schedules, and evaluation design.

## Practice Path

- derive Bellman updates by hand for a small MDP
- compare Monte Carlo and TD estimates on the same episodic environment
- test how approximation changes stability relative to the tabular baseline
- use the mini-project to reason about reward design and policy evaluation together
