# Reinforcement Learning Cheatsheets

Use the cheatsheets after the concept pages, not instead of them. They are best for recalling Bellman equations, update rules, and algorithm comparisons once the formal meaning is already clear.

## Recommended Use

- pair the cheat sheet with concept pages `02` through `06` when implementing algorithms
- use it to compare Monte Carlo, TD, policy-gradient, and actor-critic update structures
- keep the concept pages open whenever a formula's assumptions are unclear
