# TypeScript Implementations

This directory is the TypeScript counterpart to the Python implementation track. Use it for lightweight experiments, visualizations, and smaller interfaces that benefit from a web-first stack.

## Suggested Scope

- simple environment simulators for bandits and tabular RL
- browser-based visualizations for value iteration and exploration schedules
- small utilities that complement, rather than duplicate, the Python reference code
