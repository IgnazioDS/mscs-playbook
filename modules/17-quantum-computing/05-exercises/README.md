# Quantum Computing Exercises

These exercises should emphasize careful reasoning about states, measurements, and resource claims rather than slogan-level explanations.

## Suggested Practice

- compute small measurement probabilities by hand
- trace one- and two-qubit gate sequences explicitly
- critique a quantum-advantage claim by naming the hidden resource assumptions
