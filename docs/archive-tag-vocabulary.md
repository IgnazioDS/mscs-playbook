# Archive Tag Vocabulary

## Purpose

Tags should improve discovery, not become an uncontrolled keyword dump.

## Preferred Tag Families

- Archive role: `archive`, `navigation`, `blueprint`, `checklist`, `guide`
- Curriculum role: `module`, `track`, `project`, `concept`, `case-study`
- Domain tags: use stable curriculum topics such as `algorithms`, `networking`, `machine-learning`, `robotics`, `ethics`
- Use-case tags: `portfolio`, `operations`, `reliability`, `retrieval`, `evaluation`

## Rules

- Prefer 2 to 5 tags per page.
- Reuse existing vocabulary before inventing a new tag.
- Use compound slugs with hyphens, not spaces.
- Avoid adding tags that simply repeat the full page title.
- Add explicit leaf tags only where they improve search, relation quality, or future curation.
